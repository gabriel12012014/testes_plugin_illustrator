# Como Instalar o Script no Adobe Illustrator (Mac)

O `GraphMaker.jsx` é um **Script** (e não um plugin UXP padrão), o que é ótimo porque funciona em praticamente qualquer versão do Illustrator.

Você tem duas formas de usar:

## Método 1: Instalação Fixa (Recomendado)
Para que o script apareça sempre no menu do Illustrator sem precisar procurar o arquivo toda vez.

1.  Feche o Adobe Illustrator.
2.  Copie o arquivo `GraphMaker.jsx`.
3.  Vá até a pasta de Scripts do Illustrator. Geralmente fica em:
    *   `/Applications/Adobe Illustrator 2026/Presets.localized/pt_BR/Scripts`
    *   (O nome da pasta `pt_BR` pode variar dependendo do idioma do seu Mac, pode ser `en_US`, etc).
4.  Cole o arquivo lá dentro.
5.  Abra o Illustrator.
6.  O script vai aparecer em: **Arquivo > Scripts > GraphMaker** (File > Scripts > GraphMaker).

## Método 2: Uso Direto (Sem instalar)
Se você não quiser ou não puder copiar para a pasta de aplicativos.

1.  Abra o Illustrator.
2.  Vá em **Arquivo > Scripts > Outro Script...** (File > Scripts > Other Script...).
3.  Navegue até onde você salvou o `GraphMaker.jsx` e selecione ele.
4.  Pronto, ele vai rodar.
